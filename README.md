# 📸 批量实验室仪器照片调色脚本 (Lab Photo Color Grader)

这是一个基于 Python 和 OpenCV 的轻量级图像批量处理脚本。专为**实验室仪器照片**设计，通过算法模拟曲线调色，一键将照片转换为干净、专业的“偏蓝冷色调”，并支持成百上千张图片的全自动批量处理。

## ✨ 核心特性

- 🚀 **自动化批量处理**：告别手机修图软件的单张操作，几秒钟即可处理整个文件夹的照片。
- 🔵 **平滑曲线算法 (Sine LUT)**：利用正弦函数 (`np.sin`) 生成非线性的 Lookup Table (LUT)。专门针对蓝色通道的中间调进行提拉，**绝不破坏高光（纯白）和阴影（纯黑）的细节**，色彩过渡自然。
- 📁 **完美支持中文路径**：针对 Windows 环境下 OpenCV 原生不支持中文字符的痛点，使用二进制流（`cv2.imdecode` / `cv2.imencode`）底层重写了读写函数，路径命名随心所欲。
- 🛡️ **安全无损**：自动另存为新文件（自带 `blue_` 前缀），绝不覆盖原始实验数据照片。

## 🛠️ 安装与依赖

运行此脚本需要 Python 环境。请在终端（或命令行）中运行以下命令安装必要的依赖库：

```bash
pip install opencv-python numpy
```

## 🚀 快速开始

1. **克隆/下载仓库**到本地：
   ```bash
   git clone https://github.com/chujie124/python-image-script.git
   ```
2. **打开 `p_python.py`**，在代码最下方的「配置区域」修改你的文件夹路径：
   ```python
   # ================= 配置区域 =================
   INPUT_FOLDER = r"你的待处理图片文件夹路径"
   OUTPUT_FOLDER = r"你的输出图片文件夹路径"
   
   # 蓝色曲线拉高程度 (建议调整范围: 20 到 80)
   BLUE_INTENSITY = 40  
   # ===========================================
   ```
3. **运行脚本**：
   ```bash
   python p_python.py
   ```

## ⚙️ 参数调节指南

核心参数 `BLUE_INTENSITY` 决定了照片的“冷感”程度：
- **`20 - 30`**：轻微去黄，自然清新。
- **`40 - 50`**：标准的实验室冷色调（推荐默认值）。
- **`60 - 80`**：强烈的幽蓝色调，适合原本极度偏暖/偏黄的照片。
